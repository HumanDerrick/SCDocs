    通用 #

    HE+ by University of Cambridge #

    这是剑桥大学提供的一个网站，可以作为探索你感兴趣的专业的起点。类别清晰，一些主题带有简单的介绍和推荐视频。此外，还包括一些资源，如播客和 YouTube 视频链接。

    Coursera #

    这是一个网站，提供免费和付费的高级课程和在线学位，课程来自世界各地知名大学的教授。

    Horizon Inspires #

    这是一个提供不同种类的研究项目的网站。一个很好的地方，在寒暑假期间寻找课外活动。

    edX #

    这是一个由斯坦福大学开设的在线课程平台。
